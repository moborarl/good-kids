# Good Kids ⭐ — บันทึกดาวงานบ้าน

Dashboard สำหรับบันทึกค่าจ้างงานบ้านของเด็กๆ เป็น "ดาว" ให้เด็กสนุกกับการทำงานบ้าน สะสมดาว และนำดาวมาแลกของรางวัล

## ฟีเจอร์

### สำหรับเด็ก 👧👦
- โปรไฟล์ส่วนตัว เลือก Avatar และสีพื้นหลังได้เอง
- กดบันทึกงานบ้านที่ทำเสร็จ ได้ดาวทันทีพร้อม animation ⭐
- เห็น progress เป้าหมายดาวรายสัปดาห์ และยอดดาวสะสม
- แลกดาวเป็นของรางวัล 🎁
- ดูประวัติการได้ดาว / แลกของรางวัล

### สำหรับผู้ปกครอง 🔐 (ป้องกันด้วย PIN)
- เพิ่ม / แก้ไข / ลบ โปรไฟล์เด็ก
- กำหนดค่าจ้าง (จำนวนดาว) ของงานบ้านแต่ละงาน
- กำหนดราคาดาวของของรางวัลแต่ละชิ้น
- ดูสถิติรวมและประวัติทั้งหมด ลบรายการที่บันทึกผิดได้
- ตั้งเป้าหมายดาวต่อสัปดาห์ เปลี่ยนรหัส PIN
- Export / Import ข้อมูล (JSON) เพื่อย้ายระหว่างอุปกรณ์

> PIN เริ่มต้น: `1234` (เปลี่ยนได้ในแท็บตั้งค่า)

## เทคโนโลยี

- HTML / CSS / JavaScript ล้วน — ไม่มี build step, ไม่มี dependency
- ข้อมูลเก็บใน `localStorage` ของเครื่องที่ใช้งาน
- Responsive รองรับ iPhone, iPad และ Laptop
- Deploy บน Cloudflare Pages

## รันในเครื่อง

เปิด `index.html` ตรงๆ หรือใช้ static server:

```bash
npx serve .
```

## Deploy

```bash
npx wrangler pages deploy . --project-name=good-kids
```
